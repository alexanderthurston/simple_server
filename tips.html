HTML Tags

  1. <a>
    • Purpose: Anchors a hyperlink to element
    • Inline element
    • Accepts flow or phrasing content
  2. <body>
    • Purpose: Contain all media for the document
    • Neither block-level or inline
    • Accepts flow content
    • Required parent tag: Must be the second element of <html>
  3. <code>
    • Purpose: Display text in monospace font
    • Inline element
    • Accepts phrasing content
  4. <div>
    • Purpose: Separate sections of the document
    • Block-level element
    • Accepts flow content
  5. <h1>
    • Purpose: Largest header element
    • Block-level element
    • Accepts phrasing content
  6. <h2>
    • Purpose: Second largest header element
    • Block-level element
    • Accepts phrasing content
  7. <head>
    • Purpose: Contains all metadata concerning the document.
    • Neither block-level or inline
    • Accepts metadata content
    • Required parent tag: Must be the first element of <html>
  8. <img>
    • Purpose: Display an image
    • Inline element
    • Empty tag, accepts multiple image formats such as .apng, .avif, .gif, .jpg, .png, .svg, ...
  9. <li>
    • Purpose: Represent a list item
    • Block-level element
    • Accepts flow content
    • Required parent tag: Must be a child of one of the following: <ul>, <ol>, <menu>
  10. <ol>
    • Purpose: Create an ordered list of data
    • Block-level element
    • Accepts zero or more of the following: <li>, <script>, <template>
  11. <p>
    • Purpose: Represent a paragraph
    • Block-level element
    • Accepts phrasing content
  12. <title>
    • Purpose: Displays contents on browser title bar
    • Neither block-level or inline
    • Accepts text content not including inter-element whitespace
    • Required parent tag: A <head> element that does not contain another title element.
  13. <ul>
    • Purpose: Create an unordered list of data
    • Block-level element
    • Accepts zero or more of the following: <li>, <script>, <template>

HTML Entities

  1. &amp;
    • Name: Ampersand
    • Appearance: &
  2. &gt;
    • Name: Greater than symbol
    • Appearance: >
  3. &lt;
    • Name: Less than symbol
    • Appearance: <
  4. &quot;
    • Name: Quotation marks
    • Appearance: "

CSS Selectors

  1. .class
    • Purpose: Selects all elements with class = "class".
    • Ex. < h1 class="heading" >
  2. element
    • Purpose: Selects a particular HTML element
  3. #id
    • Purpose: Selects all elements with id = "id"
    • Ex. < p id="specialParagraph" >
  4. element:nth-child(n)
    • Purpose: Selects every nth child element of element
    • Ex. ol li:nth-child(2n){ /*This selects every other child beginning with the second child.*/}

CSS Properties

  1. background-color
    • Set background color of an element
    • Possible val: midnightblue, wheat
  2. color
    • Set color of element text
    • Possible val: snow, palegreen
  3. font-family
    • Sets font family for text
    • Possible val: verdana, courier new
  4. font-size
    • Sets size of text
    • Possible val: 150%, 200px
  5. font-style
    • Sets style of text
    • Possible val: italic, oblique
  6. font-weight
    • Sets the weight of text
    • Possible val: lighter
  7. line-height
    • Specifies the line height
    • Possible val: 150%, 30px
  8. margin
    • Specifies the margins
    • Possible val: 150%, 200px
  9. padding
    • Specifies an elements padding
    • Possible val: 35px
  10. text-align
    • Sets the element's text alignment
    • Possible val: center, right, justify
  11. width
    • Specifies the width of an element
    • Possible val: 150%, 200px
